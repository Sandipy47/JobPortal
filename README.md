JobPortal is a responsive, modern front-end design for a comprehensive job portal platform. It is built to provide job seekers with a seamless experience for searching, exploring statistics, and accessing dedicated login/registration portals.

The current version focuses on the visual presentation, responsive layout, and interactive elements of the core pages.

âœ¨ Features

Modern & Responsive Design: Fully responsive layout using Tailwind CSS (for the main landing page) and custom CSS, ensuring a perfect experience on desktop, tablet, and mobile devices.

Intuitive Job Search: A clear, prominent search bar allowing users to filter by keywords, location, and experience level.

Key Statistics Dashboard: Visually engaging cards showcasing platform growth, response rates, and popular job categories (Developer, Customer Service, Worldwide).

Dedicated Portals: Separate, themed pages for JobSeeker Login, Employer Login, and registration.

Dynamic Navigation & Footer: Clear navigation links and an organized footer with subscription functionality.

ğŸ’» Technologies Used

This project utilizes standard web technologies for a modern, fast, and maintainable front-end structure.

HTML5: The core structure of the web pages.

CSS3 (Custom & Embedded): Used for deep styling and visual appeal.

Tailwind CSS: Employed on the main landing page (index.html) for utility-first styling and responsiveness.

Bootstrap CSS: Included in auxiliary pages (login.html, register.html, employer.html) for structure.

Bootstrap Icons: Used for professional and recognizable iconography across the site.

Vanilla JavaScript: Minimal scripting for handling form submission events.
